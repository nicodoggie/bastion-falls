2. Sole argument of intransitive verbs

So in "Where is the store?", both elements form a single equated argument, taking absolutive marking:

```plaintext
'atal-es matiltel-es?
where.PROX-ABS store-ABS
"Where is the store?"
```

The ergative would only be used if the interrogative word is acting as the agent of a transitive verb, as shown in the example "'alis-el materok-es ward-'er?" (lines 897-899).
